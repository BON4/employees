# Empyee tree service implementation
To start, configure constants (no .cfg yet).

## Install
```
> go get github.com/BON4/employees
```

## Run
```
> cd employees\cmd\app
employees\cmd\app> go build .
employees\cmd\app> .\app
```

## Test
```
employees> go test ./...
```
